# home-pgemu-journal

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容多样，不针对任何特定域名或网站，主要作为个人项目资料的整理与公开存放。

## 目录说明

```
.
├── index.html          # 主索引页（可浏览所有归档页面）
├── pages/              # 存放各独立 HTML 页面
│   ├── example.html
│   └── ...
└── assets/             # 页面引用的公共资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的入口页面，列出所有归档页面链接。
- `pages/`：每个独立 HTML 文件放在此目录下，文件名与页面内容对应。
- `assets/`：存放页面间共享的静态资源，按类型分目录管理。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面内容以个人学习、记录或实验为目的，不构成正式发布或商业用途。
- 每个页面独立可访问，可通过 `index.html` 或直接打开文件路径浏览。
- 部分页面可能引用外部资源（如字体、图标库），请确保网络连接正常。

## 维护说明

- 本仓库不定期更新，新增页面或修改已有内容。
- 如有页面失效或资源链接错误，欢迎提交 Issue 或 Pull Request。
- 不建议将本仓库用于生产环境或重要数据存储。
- 所有内容遵循开源许可协议（见 LICENSE 文件），使用时请保留原始出处信息。
